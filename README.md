# 📝 React To-Do List App

A simple To-Do List app built with React.js and Tailwind CSS.  
It uses localStorage to save tasks permanently in the browser.

---

## 🚀 Features

- Add new tasks  
- Delete tasks  
- Mark tasks as completed  
- Data saved in localStorage  
- Clean and responsive UI  

---

## 🛠️ Tech Stack

- React.js  
- Tailwind CSS  
- Vite  
- JavaScript  
- LocalStorage API  

---

## 💾 How Local Storage Works

- Tasks are saved in browser storage  
- Data stays after refresh  
- No backend needed
